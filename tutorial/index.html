Score チュートリアル

映像制作スタジオ向け統合プロジェクト管理システム

📺 Score とは

Score は 映像制作スタジオ向けの統合プロジェクト管理システム です。プロジェクト・SHOT・task・asset・QC・thread を 1 つの画面に集約し、 PM・Director・Lead・User など各ロールが 連絡漏れなく 業務を進められます。

  • ✓SHOT 関係者 thread 自動作成で連絡集約
  • ✓QC 依頼 → Director 判定 → Retake 発令 の完全動線
  • ✓Push + SSE + 画面 badge の三重通知で見逃し防止
  • ✓朝のルーティン + 退勤報告で稼働可視化

🎭 ロール別 1 日業務動線

各ロールを選択して、Score を使った 1 日の業務動線を確認してください。各ステージは「目的・結果・ポイント・💬 例えば」 で整理されており、初めての方でも 1 日の流れが具体的に理解できます。

📋

PM (プロダクションマネージャー)

担当: 田中 太郎

  • ▸全プロジェクトの進捗・納品スケジュール 集約把握
  • ▸全 SHOT 関係者 thread に自動 participants として含まれる (連絡漏れ防止)
  • ▸納品チェックリストで全 SHOT が Approved 済か確認してから提出
🎬

Director (作業指示・QC業務)

担当: 山田 博

  • ▸QC ビューアで Approve/Retake 判定 (Director 専用権限)
  • ▸Retake 発令 page で 作業指示 (全体方針・マーカー・参考素材) を User に具体的に伝達
  • ▸全 SHOT QC 状況を Director ダッシュボードで集約把握 → 判定順を効率化
💡

Lead (テクニカルサポート)

担当: 加藤 健司

  • ▸User からの技術相談・トラブル対応を SHOT thread で支援
  • ▸Director QC で発生した テクニカル issue を User と共同解決
  • ▸リファレンス・過去 asset・pipeline 情報の参照支援
🖥️

User (一般業務担当・Animator/FX 等)

担当: 佐藤 花子

  • ▸担当 SHOT の作業着手・asset upload・部内 Review/QC 依頼提出
  • ▸Retake 発令時 リファレンス・マーカー・コメントを /retake_view で一覧確認
  • ▸shot_detail で 自分の担当 task と asset version 履歴を一括管理

共通操作: ログイン〜ログアウト

  1. ブラウザで Score URL を開き、メールアドレス + パスワードでログイン
  2. (前日退勤未提出時) 退勤報告 page で前日分提出 → ルーティン page へ
  3. 朝のルーティン (体調確認) 提出 → ダッシュボードへ
  4. サイドメニューから各機能に移動
  5. 退勤時は「退勤報告」を提出してからログアウト
