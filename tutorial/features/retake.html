🔁 Retake 発令

📘 この機能の目的

Director が Retake を発令する画面 + 受信側の view-only リテイク確認 page。video player・タイムラインマーカー・コメント・オーバーペイント・参考素材添付を統合。

📸 画面 (6 枚)

① 入力 page 全体 (動画 + 全体方針 + 参考素材)

/director_retake_input 全体図 — 動画 player が表示され、 リファレンス画像が添付された状態

② フレームプレビュー (実 video 表示)

実 mp4 video player + TC overlay (現在の playhead 表示) + 再生中状態

③ オーバーペイント機能

video フレーム上に色付き drag 描画で修正箇所をハイライト + マーカー保存

④ タイムライン + マーカー

タイムライン scrubber + 保存済マーカー (例 0:12 桜並木) が複数並ぶ状態

⑤ 参考素材添付 (drag&drop 状態)

リファレンス画像 / 動画を drag&drop で添付した直後の thumb preview 表示

⑦ 受信側 /retake_view (User)

User (佐藤) が受信する view-only page — パンくず + 対象 video (S1) + マーカー button 3 件 + コメント card + メタ (発令者 山田 / 期日) + 全体方針 + 参考素材 2 枚 inline + task thread link

📍 シナリオ: 山田 Director が桜咲く街 sq01 c05 v04 を Retake 発令 → 「桜花ピンク 10% 強化」 全体方針 + 0:12 マーカー + 参考画像 2 枚添付 → 佐藤 User が /retake_view で確認 → 修正着手。

✨ 主な機能

  • /director_retake_input (Director 発令 page)
  •   ・フレームプレビュー = 実 video player (mp4 自動再生・playhead↔video.currentTime 双方向 sync)
  •   ・オーバーペイント canvas (drag 描画・色切替・マーカー保存 + 復元)
  •   ・タイムライン scrubber (click/drag で playhead 移動)
  •   ・参考素材添付 (image/video file 選択 + drag&drop + URL)
  •   ・期日 default = 今日
  •   ・プレビュー block 動的更新 (input 即時反映)
  •   ・🚀 リテイクを発行 → /api/bff/retakes (multipart) → SHOT thread 「🔁 Retake 発令」 投稿
  • /retake_view/{shot_id}/{task_id} (受信 view-only page)
  •   ・対象 asset 再生 (TC overlay リアルタイム + マーカー TC click でシーク + 自動 play)
  •   ・メタ (優先度 / 期日 / 発令者名 / 発令日時)
  •   ・全体方針 (Director 入力 direction 全文)
  •   ・コメント card click → video seek + play
  •   ・参考素材 inline 表示 (画像拡大 / 動画 player / URL link)
  •   ・💬 task thread を見る link (messages 連動)

🌸 使用例 — 架空 project「桜咲く街」

山田 Director が桜咲く街 sq01 c05 v04 を Retake 発令 → 「桜花ピンク 10% 強化」 全体方針 + 0:12 マーカー + 参考画像 2 枚添付 → 佐藤 User が /retake_view で確認 → 修正着手。

🛠 操作方法

  1. Director: qc_viewer → 🔁 Retake link → /director_retake_input?shot_id&task_id&as_role=director
  2. 全体方針 textarea 入力 + 参考素材添付 (drag&drop or click) + 優先度 + 期日
  3. video player でマーカー保存 + コメント記入 → プレビュー block 即時反映
  4. 🚀 リテイクを発行 → 「✅ 発行完了 — thread / push / sse」 表示
  5. User (受信側): 通知 toast → SHOT thread → ▶ ボタン → /retake_view/{shot_id}/{task_id}
  6. video + マーカー button + コメント card で全内容確認 → 修正作業へ
