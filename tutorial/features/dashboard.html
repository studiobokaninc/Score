🏠 ダッシュボード

📘 この機能の目的

ログイン後 最初に表示される統合 home 画面。自分担当 task・受信 QC 依頼・通知件数・出勤時刻 を 1 画面で把握する。

📸 画面

📍 画面例: 佐藤 User が桜咲く街 sq01 c05 担当・朝ログイン後 Dashboard で「Retake 1 件・担当 task 3 件・出勤 09:43」 を確認。受信 QC 依頼 card click → qc_viewer 遷移して内容確認。

✨ 主な機能

  • 出勤時刻 chip — 朝のルーティン提出時刻が🕘 HH:MM で常時表示 (殿御命 2026-06-05)
  • 本日やるべきタスク — 自分担当 task を優先度順 (qc_fb→qc→mk→wip) 表示
  • 📥 受信 QC/Review 依頼 — SHOT thread 経由の連絡も統合表示・thread と公式 status の両方をカバー
  • QC 依頼 link — qc_viewer (/qc/{shot_id}?task_id&asset_id) へ 1 click 遷移
  • Calendar 今日のイベント — 試写・mtg・納品日が時刻順に表示
  • upcoming milestones — 7 日以内の重要 deadline 浮上

🌸 使用例 — 架空 project「桜咲く街」

佐藤 User が桜咲く街 sq01 c05 担当・朝ログイン後 Dashboard で「Retake 1 件・担当 task 3 件・出勤 09:43」 を確認。受信 QC 依頼 card click → qc_viewer 遷移して内容確認。

🛠 操作方法

  1. ログイン → routine 提出 → Dashboard 自動遷移
  2. 「📥 受信 QC 依頼」 紫 card click → qc_viewer で対象 asset 視聴
  3. 「本日のタスク」 行 click → SHOT 詳細 page へ
  4. 「upcoming milestones」 で 1 週間先までの重要日把握
