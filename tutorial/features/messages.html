💬 Messages (メッセージ)

📘 この機能の目的

SHOT 関係者 thread (PM/Director/Lead/User) と 1対1 DM を統合表示。連絡履歴を SHOT 単位で集約。

📸 画面

📍 画面例: 佐藤 User が桜咲く街 sq01 c05 QC 依頼提出 → 自動的に PM 太郎・山田 Director・加藤 Lead を含む thread 作成 → 山田 Director から「Lighting 微調整希望」 返信 → thread 内で会話継続。

✨ 主な機能

  • SHOT 関係者 thread 自動作成 — QC/Review/Retake 提出時に PM + Director + Lead + assignees で構成
  • thread 階層 title — 「c05 Compositing」 等の SHOT + task 名で識別容易 (殿御命 2026-06-05)
  • URL ?thread=N で自動 open — 通知から直接該当 thread へ
  • QC ビューア URL は緑ボタン化 — 本文中の link が即 click 可能
  • SHOT/DM 振り分け — content から自動判別 (QC/Review/Approved/Retake で始まる = SHOT)
  • 他 page で thread link click → 右スライドイン drawer (殿御命 2026-06-05)

🌸 使用例 — 架空 project「桜咲く街」

佐藤 User が桜咲く街 sq01 c05 QC 依頼提出 → 自動的に PM 太郎・山田 Director・加藤 Lead を含む thread 作成 → 山田 Director から「Lighting 微調整希望」 返信 → thread 内で会話継続。

🛠 操作方法

  1. サイドメニュー💬 → Messages
  2. SHOT タブ / DM タブ切替
  3. thread 行 click → 会話履歴表示
  4. 本文中の緑 button → qc_viewer 即遷移
  5. 通知から /messages?thread=280052 で直接 open
