📅 Calendar (スケジュール)

📘 この機能の目的

スタジオ全体のマイルストーン・試写・納品日・mtg を月/週/日 ビューで可視化。リソース競合・日程過密を早期発見。

📸 画面 (2 枚)

① Calendar 月/週ビュー

佐藤 花子 のスケジュール — Meeting / Milestone / Deadline / Workshop / Task / General の filter 切替 + 月-週-日ビュー

② イベント詳細 modal (mtg リンク)撮影待ち

イベント click → modal 表示 — 時刻 / 場所 / 参加者 + 🎥 Zoom / Teams / Meet など mtg link button (殿御命 2026-06-08 改修)

📍 シナリオ: PM 太郎が桜咲く街 第 4 話 試写日 (6/15) と 別 CM 案件納品日 (6/16) が近接していることを発見 → 加藤 Lead に Lighting 完了予定を 1 日前倒し依頼。

✨ 主な機能

  • 3 ビュー切替 — 月・週・日で表示密度切替
  • color 分け — マイルストーン (青) / 試写 (黄) / 納品日 (赤) / mtg (緑)
  • イベント click → 詳細 popup — 関連プロジェクト link 付き
  • 予定追加 + 参加者選択 — mtg 招待・自動 thread 作成
  • nibu Calendar BE 連携 — Score Calendar 機能は nibu 殿御陣営 BE に依存

🌸 使用例 — 架空 project「桜咲く街」

PM 太郎が桜咲く街 第 4 話 試写日 (6/15) と 別 CM 案件納品日 (6/16) が近接していることを発見 → 加藤 Lead に Lighting 完了予定を 1 日前倒し依頼。

🛠 操作方法

  1. サイドメニュー📅 → Calendar
  2. 月/週/日 button で切替
  3. イベント click → popup で詳細 + プロジェクト link
  4. 「予定追加」 button → mtg 招待 + 参加者選択
