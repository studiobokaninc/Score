🔔 通知センター

📘 この機能の目的

Push / SSE / 画面 badge の三重通知を一元表示。category 別 (qc_request/review_request/retake/approve) で履歴管理。

📸 画面

📍 画面例: 桜咲く街 sq01 c05 Approve 通知が Push toast + 画面 badge + 通知センター 3 経路で佐藤 User に届く → 通知 click → SHOT thread へ遷移。

✨ 主な機能

  • 三重立て通知 — Push (VAPID/SW root scope) + SSE (3 秒 keepalive) + 画面 badge
  • category 別 filter — qc_request / review_request / retake / approve / mention
  • 既読/未読管理 — PATCH /api/bff/notifications/{id}/read
  • SSE in-page badge — 在席時のみ受信・bandwidth 節約
  • Push 有効化 button — notif_settings から SW 登録

🌸 使用例 — 架空 project「桜咲く街」

桜咲く街 sq01 c05 Approve 通知が Push toast + 画面 badge + 通知センター 3 経路で佐藤 User に届く → 通知 click → SHOT thread へ遷移。

🛠 操作方法

  1. サイドメニュー🔔 → 通知センター
  2. category filter で絞込み
  3. 通知 click → 該当 page (qc_viewer / messages 等) へ
  4. 全既読 button で一括 mark
